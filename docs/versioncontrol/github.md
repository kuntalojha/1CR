# GitHub

- GitHub is a code hosting platform for version control and collaboration.

## Some GitHub Comments

- `git rmote origin master` - Remove a remote repository.
- `git remote add origin https://github.com/kuntalojha/1CR.git` - Add a remote repository.
- `git remote rm origin` - Remove a remote repository.
- `git remote -v` - List all remote repositories for the current repository.
